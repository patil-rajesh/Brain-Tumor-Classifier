🧠 Brain Tumor Classifier Model
The Brain Tumor Classifier Model is a professional-grade medical imaging dashboard designed to classify brain tumors from MRI scans. It combines a deep learning Convolutional Neural Network (CNN) with an intuitive, high-fidelity interface to provide automated diagnostic support and model interpretability.

✨ Key Features
Four-Class Classification: Accurately detects and categorizes scans into Glioma, Meningioma, Pituitary Tumor, or No Tumor.

Explainable AI (Grad-CAM): Generates heatmaps to visualize the specific regions of the MRI that the model used to make its prediction, ensuring transparency in clinical decision-making.

Performance Analytics: Displays real-time evaluation metrics (Accuracy, F1-Score, Precision) and interactive training history curves.

Figma-Inspired UI: A modern, dark-themed dashboard built with Streamlit, optimized for clarity and ease of use.

Multiple Input Modes: Support for direct file uploads, a sample gallery for testing, and a random demo generator.

🛠️ Technical Stack
Model Backend: TensorFlow / Keras (CNN Architecture)

Frontend UI: Streamlit

Computer Vision: OpenCV, Pillow (PIL)

Data Visualization: Plotly, Matplotlib

Deployment: Streamlit Community Cloud

🚀 Getting Started
1. Prerequisites

Ensure you have Python 3.10 or higher installed.

2. Installation

Clone the repository and install the required dependencies:

Bash
git clone https://github.com/YOUR_USERNAME/brain-tumor-classifier.git
cd brain-tumor-classifier
pip install -r requirements.txt
3. Running the App

Launch the dashboard locally:

Bash
streamlit run app.py
